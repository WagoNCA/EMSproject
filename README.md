# EMS / REST API Project

This is a small project that I do to quickly learn the basics of go / docker / adminer / echo / influxdb.

## Run the project

* Open the terminal
* Go to the EMSproject folder
* Use docker compose up (to start the services)
* Run the main.go file (to start the local host)
* In any browser, go to [this](https://localhost:8000) link

You can now use the methods to work with the EMS tool.
